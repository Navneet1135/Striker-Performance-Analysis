# ⚽ Segmenting and Classifying the Best Strikers

## Project Overview

This project analyzes football striker performance using Data Analytics and Machine Learning techniques. The project includes data cleaning, statistical analysis, feature engineering, clustering, and predictive modeling to identify performance patterns among football strikers.

The main objective is to classify players into:

- Best Strikers
- Regular Strikers

using player performance metrics such as goals scored, assists, shot accuracy, dribbling success, consistency, hold-up play, and defensive contribution.
